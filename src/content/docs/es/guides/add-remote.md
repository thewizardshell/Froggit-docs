---
title: Agregar Remoto
description: Cómo agregar un nuevo repositorio remoto en Froggit.
---

La vista de Agregar Remoto te permite configurar una nueva conexión a un repositorio remoto.

## Campos

El formulario tiene dos campos:

**Nombre del remoto.** El identificador local para el remoto (por ejemplo, `origin`, `upstream`).

**URL del remoto.** La dirección del repositorio remoto (por ejemplo, `https://github.com/usuario/repo.git` o `git@github.com:usuario/repo.git`).

## Controles

| Tecla | Acción |
|-------|--------|
| `Tab` | Cambiar entre los campos de nombre y URL |
| `Enter` | Confirmar o pasar al siguiente campo |
| `Backspace` | Borrar un carácter |
| `Esc` | Cancelar |
