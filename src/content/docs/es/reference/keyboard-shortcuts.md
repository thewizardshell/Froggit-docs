Esta página lista todos los atajos de teclado disponibles en Froggit, organizados por la vista donde aplican.

## Global

Estos atajos funcionan en la mayoría de las vistas.

| Tecla | Acción |
|-------|--------|
| `q` | Salir de Froggit |
| `Ctrl+C` | Salir de Froggit |
| `Esc` | Volver atrás o cancelar |
| `?` | Mostrar ayuda |

## Vista de Archivos (Modo Normal)

La vista principal donde gestionas tus archivos.

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar archivos |
| `Espacio` | Hacer staging o unstaging del archivo seleccionado |
| `a` | Hacer staging de todos los archivos |
| `u` | Hacer unstaging de todos los archivos |
| `d` | Abrir vista previa de diff del archivo seleccionado |
| `x` | Descartar cambios del archivo seleccionado |
| `c` | Abrir vista de commit (requiere archivos en staging) |
| `r` | Actualizar estado de archivos |
| `f` | Fetch desde el remoto |
| `l` | Pull desde el remoto |
| `p` | Push al remoto |
| `b` | Abrir vista de ramas |
| `m` | Abrir vista de remotos |
| `A` | Entrar al Modo Avanzado |
| `?` | Mostrar ayuda |

## Vista de Archivos (Modo Avanzado)

Se activa presionando `A` desde la Vista de Archivos.

| Tecla | Acción |
|-------|--------|
| `L` | Abrir gráfico de log |
| `M` | Abrir vista de merge |
| `R` | Abrir vista de rebase |
| `S` | Abrir gestor de stash |
| `Esc` | Salir del Modo Avanzado |
| `?` | Mostrar ayuda |

## Vista de Commit

| Tecla | Acción |
|-------|--------|
| `Enter` | Confirmar y crear el commit |
| `Backspace` | Eliminar un carácter |
| `Tab` | Generar mensaje de commit con AI (cuando Copilot está disponible) |
| `Esc` | Cancelar y volver a la Vista de Archivos |

## Vista de Ramas

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar ramas |
| `Enter` | Cambiar a la rama seleccionada |
| `n` | Crear una nueva rama |
| `d` | Eliminar la rama seleccionada |
| `Esc` | Volver a la Vista de Archivos |
| `q` | Salir |

## Vista de Remotos

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar remotos |
| `n` | Agregar un nuevo remoto |
| `d` | Eliminar el remoto seleccionado |
| `Esc` | Volver a la Vista de Archivos |
| `q` | Salir |

## Vista de Agregar Remoto

| Tecla | Acción |
|-------|--------|
| `Tab` | Cambiar entre campos de nombre y URL |
| `Enter` | Confirmar o pasar al siguiente campo |
| `Backspace` | Eliminar un carácter |
| `Esc` | Cancelar |

## Vista de Nueva Rama

| Tecla | Acción |
|-------|--------|
| `Enter` | Crear la rama |
| `Backspace` | Eliminar un carácter |
| `Esc` | Cancelar |

## Vista de Merge

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar ramas |
| `Enter` | Seleccionar una rama |
| `M` | Confirmar el merge |
| `P` | Continuar después de resolver conflictos |
| `X` | Abortar el merge |
| `Esc` | Cancelar |

## Vista de Rebase

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar ramas |
| `Enter` | Seleccionar una rama |
| `R` | Confirmar el rebase |
| `P` | Continuar después de resolver conflictos |
| `X` | Abortar el rebase |
| `Esc` | Cancelar |

## Vista de Stash

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar stashes |
| `Enter` | Aplicar el stash seleccionado |
| `p` | Pop del stash seleccionado (aplicar y eliminar) |
| `d` | Eliminar el stash seleccionado |
| `v` | Ver el contenido de un stash |
| `s` | Guardar cambios actuales como nuevo stash |
| `Esc` | Volver a la Vista de Archivos |
| `?` | Mostrar ayuda |

## Vista de Mensaje de Stash

| Tecla | Acción |
|-------|--------|
| `Enter` | Guardar el stash |
| `Backspace` | Eliminar un carácter |
| `Esc` | Cancelar |

## Vista de Diff

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Desplazarse por el diff |
| `Esc` | Volver a la Vista de Archivos |

## Vista del Gráfico de Log

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar commits |
| `Esc` | Volver a la Vista de Archivos |

## Vista de Lista de Repositorios

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar repositorios |
| `c` | Clonar el repositorio seleccionado |
| `Esc` | Volver atrás |

## Vista de Quick Start

| Tecla | Acción |
|-------|--------|
| `Arriba` / `Abajo` | Navegar opciones |
| `Enter` | Seleccionar una opción |
| `1` / `2` / `3` | Selección rápida de una opción |
| `q` | Salir |

## Diálogo de Confirmación

| Tecla | Acción |
|-------|--------|
| `y` | Confirmar |
| `n` | Cancelar |
