Welcome to the official Froggit roadmap.  
Here we track completed features, what's in progress, and our goals for version 1.0.0.

---

## ✅ Completed Features

- ✔ Git log viewer (clean, readable history)
- ✔ Merge support with conflict detection
- ✔ Rebase flow (interactive)
- ✔ GitHub CLI integration (requires `gh` installed)
- ✔ Stash management (create, list, drop)

---

## 🚧 In Progress

- ☐ Automated tests (for pull requests, merge/rebase)
- ☐ Configurable TOML file (colors, branding, keybindings)
- ☐ Self-update via `--update` flag
- ☐ Optional Vim-style keybindings
- ☐ Improved error handling and fallback support

---

## 🎯 Goals for v1.0.0

- Full support for essential Git operations (log, commit, branch, merge, rebase, stash)
- Optional customization (theme colors, shortcuts, branding)
- Lightweight and intuitive interface
- Zero-crash behavior under normal usage

---

## 📬 Community & Contribution

- Feature suggestions and feedback are welcome via GitHub Issues
- Polls and votes on new feature priorities will be available
- Community contributions to translations, UI, and docs are encouraged
