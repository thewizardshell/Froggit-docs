---
title: Advanced Mode
description: Access merge, rebase, stash, and log graph operations through Advanced Mode.
---

Advanced Mode is a secondary control layer in the File View that gives you access to operations beyond everyday staging and committing. It keeps the main interface clean while making powerful Git features available when you need them.

## Entering Advanced Mode

From the File View, press `A` to enter Advanced Mode. The controls bar at the bottom of the screen changes to show the advanced operations. A hint labeled `A advanced` is always visible in the normal controls bar so you know this mode exists.

## Available Operations

While in Advanced Mode, the following shortcuts become available:

| Key | Action |
|-----|--------|
| `L` | Open the [Log Graph](/advanced/log-graph) view |
| `M` | Open the [Merge](/advanced/merge) view |
| `R` | Open the [Rebase](/advanced/rebase) view |
| `S` | Open the [Stash](/advanced/stash) manager |
| `Esc` | Exit Advanced Mode and return to normal controls |
| `?` | Show help |

## Exiting Advanced Mode

Press `Esc` to return to the normal File View controls. The standard shortcuts for staging, committing, pushing, and navigating will be restored.
